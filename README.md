# Single Point of Failure (SPOF)

A Sierra-style comedy adventure. Start with `CLAUDE.md` (conventions + milestones),
`GDD.md` (the design), and `data/schema.md` (content format).

    npm install
    npm run dev      # local dev
    npm run build    # static build in dist/ -> deploy to outagequest.com

New backgrounds: `python3 tools/treat_bg.py <source.png> assets/backgrounds/<room_id>.png`
